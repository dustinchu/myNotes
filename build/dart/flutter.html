# Stack overflow Link。

  • 假設您的服務器期望username:password組合將其編碼為UTF-8（有關更多詳細信息，請參閱RFC 7617），然後使用：
import 'dart:convert';

import 'package:http/http.dart';

main() async {
  String username = 'test';
  String password = '123£';
  String basicAuth =
      'Basic ' + base64Encode(utf8.encode('$username:$password'));
  print(basicAuth);

  Response r = await get('https://api.somewhere.io',
      headers: {'authorization': basicAuth});
  print(r.statusCode);
  print(r.body);
}
